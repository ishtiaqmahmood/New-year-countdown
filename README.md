# New Year Countdown

A sleek, responsive landing page that counts down the days, hours, minutes, and seconds until the start of the next year.



## Features

- **Real-time Countdown**: Accurate calculation of time remaining until January 1st.
- **Dynamic Background Year**: Automatically updates to show the upcoming year.
- **Responsive Design**: Optimized for various screen sizes, from mobile phones to desktops.
- **Smooth Animations**: Includes a fade-in effect for the countdown timer after loading.
- **Loading State**: Displays a spinner during initial load for a polished user experience.
- **Accessibility**: Built with ARIA roles and semantic HTML for better screen reader support.

## Technologies Used

- **HTML5**: Semantic structure and metadata.
- **CSS3**: Custom properties (variables), Flexbox, and Keyframe animations.
- **JavaScript (ES6+)**: DOM manipulation and time calculation logic.

## Getting Started

### Prerequisites

You only need a modern web browser to run this project.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/new-year-countdown.git
   ```
2. Navigate to the project directory:
   ```bash
   cd new-year-countdown
   ```
3. Open `index.html` in your favorite browser.
